1) в консоли gh auth login
2) cd путь 
или если нет на компе 
git clone url_адрес
3) git branch имя_ветки
4) git checkout имя_созданной ветки (Переключение между ветками)
5) git add .
6) git commit -m "Текст к коммиту"
7) git merge сливаемая_ветка (если что-то не так, то разрешить конфликт)
8) git push
9) на сайте git hub создать pull requests 
Еще список комманд
git status
git log
git revert (нужна для удаления ненужных коммитов)
